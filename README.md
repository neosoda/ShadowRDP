# ShadowRDP

Petit ensemble de scripts PowerShell/outils pour faciliter le déploiement et l’assistance autour de RDP (Shadow / Remote Assistance).

## Contenu

- `Deploy-RDPShadow-GPO.ps1` : déploiement/paramétrage via GPO.
- `RemoteDesktopAssistantV1.ps1` / `RemoteDesktopAssistantV1.2.ps1` : scripts d’assistance.
- `Remoteicon.ico` : icône.
- `Remote_Desktop_Assistant_Documentation.pdf` : documentation.

## Pré-requis

- Windows + PowerShell (Windows PowerShell 5.1 ou PowerShell 7+).
- Droits adaptés (GPO / admin) selon l’usage.
